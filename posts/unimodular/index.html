Unimodular Matrices and Loops

2022-03-02

A unimodular matrix is a square integer matrix with determinant ±1\pm 1±1. With some basic linear algebra lemmas, we can show that a square integer matrix is unimodular if and only if its inverse exists and the inverse matrix is also an integer matrix.
